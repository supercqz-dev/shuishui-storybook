# 项目圣经 (Bible) ——常驻资产

这个目录存放整个项目的"恒定知识"：画风、世界观、所有角色。
**每本绘本都从这里读，不重复定义。** 修改这里的内容会影响所有后续生成。

## 目录结构

```
bible/
├── world.yaml          世界观（原创拟人动物世界）
├── style.yaml          画风圣经（原创现代 3D 动画风）
└── characters/
    ├── shuishui.yaml   水水（主角，白兔子）
    ├── papa.yaml       爸爸（小熊猫 Red Panda；旧红狐版归档在 _archive/papa-fox-2026-05-29/）
    ├── mama.yaml       妈妈（灰兔子）
    └── laolao.yaml     姥姥（绵羊）
```

## 字段约定

每个 YAML 包含两类字段：
- **中文描述字段**：给人看、给 LLM 写故事时参考
- **`prompt_anchor`**：英文，直接拼进图像生成 prompt（图像模型对英文 prompt 更稳）

## 修改原则

- 角色外观锁（体型、服饰、显著特征）一经确定，**不要随意改**——改了之后老绘本会和新绘本视觉不一致
- 性格、口头禅、喜好这类字段可以随时丰富（影响故事，不影响视觉）
- 修改后用 `tools/validate_bible.py`（待建）跑一遍校验
